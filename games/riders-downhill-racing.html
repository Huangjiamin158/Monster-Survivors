Game Description

Riders Downhill Racing is a bike racing game that you can play by yourself or with your friend locally. There are three game modes available: Racing Mode, Career Mode, and Free Style Mode. Try to be the first to reach the finish line in the Racing Mode. Reach the finish line before the time runs out in the Career Mode. Have a super fun ride, and don't forget to collect diamonds in the Free Style Mode to unlock more items. Fly through the ramps, do stunts, and experience exciting moments! Go to the garage to buy new bikes, costumes, ATV bikes, cross bikes, and speed boats for more great adventures! Are you ready to be the best riders? Let the ride begin!

Game Controls

Player 1 → WASD / arrow keys = move
Q = jump (Free Style mode only)
E = parachute (Free Style mode only)
Tab = switch vehicle (Free Style mode only)
Shift = NOS (not available for bike)
C = change camera view
R = reset position
Player 2 → Arrow keys = move
I = jump (Free Style mode only)
P = parachute (Free Style mode only)
Enter = switch vehicle (Free Style mode only)
M = NOS (not available for bike)
K = change camera view
U = reset position
